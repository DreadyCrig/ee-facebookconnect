Installation

Requirements #

Solspace Facebook Connect meets the same requirements as listed on the ExpressionEngine Requirements page.

  • ExpressionEngine 3.3.x - 3.4.x
  • PHP 5.3.10 or later
  • cURL
  • MySQL 5.0.3 or later
  • Windows and OS X browsers:
    • Chrome 21 or later
    • Firefox 28 or later
    • Safari 9.0 or later
    • Internet Explorer 11 or later
    • Microsoft Edge

Installing Instructions #

  1. Within the /system/user/addons/ directory, copy the fbc folder into your sites ./system/user/addons/ directory.
  2. Within the /themes/user/ directory, copy the fbc folder into your sites ./themes/user/ directory.
  3. Go to the Add-On Manager area in the ExpressionEngine control panel, scroll to down to the Third Party Add-Ons section, and find Facebook Connect in the list, and click Install.
  4. Setup and configuring:
    • To use this add-on, you'll need to create a Facebook App.
    • The Control Panel area allows you adjust preferences and run diagnostics.
    • Be sure to read over the Overview & Important Notes and Behind the Scenes documentation for this add-on.
    • If you wish to install the demo templates, click on the Demo Templates tab inside the control panel area.

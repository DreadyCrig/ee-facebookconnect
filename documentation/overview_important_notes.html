Important Notes

Below are some important things you need to know about Solspace Facebook Connect (FBC):

  • As a security provision, no one can ever login to a Super Admin account using FBC. This can be overridden at the code level by adventurous developers.
  • FBC will NOT work on a site that is not accessible to the internet. Facebook needs to be able to talk to your site. If your site is hosted on a local machine, hidden to the world, is password protected or otherwise offline, you will not be able to test.
  • Absolutely NO Facebook profile data can be synced over from Facebook into your members' profile fields, as this is strictly prohibited by Facebook. If correct permissions are set and allowed by the user, only EE screen name and email will be populated with Facebook data.
  • Facebook restricts news feed / posts to 10 per user per site per day.
  • In order for someone to login to your EE site through Facebook, they have to click the Facebook login button. This button opens a JS pop-up window that contains an Facebook login page served by Facebook itself. There is no other login method.
  • In order for the Facebook login button to work, you have to add some javascript code just above the </body> tag of your page. Facebook Connect includes a simple tag that does the work for you. See Prepare_Page documentation for more info.
  • In order for FBC to work on your site, you have to create a Facebook App for your site on Facebook. Once you register your site you will be provided with an App ID and an App Secret key. Both of these must be entered into your Facebook Connect control panel.
  • The Facebook Connect control panel includes a diagnostics section that helps you verify that a valid connection has been made between your site and Facebook.
  • Do not spam users. Your use of email must comply both with Facebook policies and with the CAN-SPAM Act.

Overview of Setup

In summary, you need to complete these steps when installing Solspace Facebook Connect:

  1. Install Solspace Facebook Connect.
  2. Create a Facebook App for your site on Facebook.
  3. Copy and past the App ID and App Secret key from Facebook into your Facebook Connect control panel.
  4. Place the Prepare_Page tag in the necessary site templates.
  5. Add Facebook Connect login buttons to your site page(s) as necessary.

Installation

  1. Within the /system/expressionengine/third_party/ directory, copy the fbc folder into your sites ./system/expressionengine/third_party/ directory.
  2. Within the /themes/third_party/ directory, copy the fbc folder into your sites ./themes/third_party/ directory.
  3. Go to the Modules area in the ExpressionEngine control panel, find Facebook Connect in the list, and click Install.
  4. Setup and configuring:
    • To use this add-on, you'll need to create a Facebook App.
    • The Control Panel area allows you adjust preferences and run diagnostics.
    • Be sure to read over the Overview & Important Notes and Behind the Scenes documentation for this add-on.
    • If you wish to install the demo templates, click on the Demo Templates tab inside the control panel area.

Updating

  1. Within your sites ./system/expressionengine/third_party/ directory, delete the fbc folder and upload the new one from the /system/expressionengine/third_party/ directory in your download package.
  2. Within your sites ./themes/third_party/ directory, delete the fbc folder and upload the new one from the /themes/third_party/ directory in your download package.
  3. Go to the Modules area in the ExpressionEngine control panel and click the Run Module Updates button.
